## Fonctionnalités
* **Interface intuitive** : Visualisation en temps réel de la tête de lecture et du ruban.
* **Exemples intégrés** : Inclut des algorithmes classiques (Incrémentation binaire, Addition unaire, etc.).
* **Table de transitions modifiable** : Créez et testez vos propres programmes.
* **Mode "Exécution automatique"** : Contrôlez la vitesse de lecture.

## Utilisation
1. Chargez un exemple depuis le menu déroulant.
2. Cliquez sur "Exécuter" ou utilisez le mode "Pas à pas".
3. Modifiez la table de transitions pour créer vos propres règles.
